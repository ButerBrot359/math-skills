## **How to run the project**

1. Clone github repo
2. ./math-skills
3. You have 2 options to run this project:
   - Use Cargo
     1. cargo run data.txt
   - Use command line
     1. cargo build
     2. ./target/debug/math-skills data.txt

### _Be sure that `data.txt` file exist in `math-skills` directory_
